# Traffic Light Color
## Description
This project is a simple python program that demonstrates the use of if-elif-else statement in Python. It is used to check multiple conditions in a sequence and execute the block of code for the first condition that evaluates to **True**.
## Syntax
```python
if condition1:
   # Code block to execute if condition is True
elif condition2:
   # Code block to execute if condition is True
elif condition3:
   # Code block to execute if condition is True
else:
   # Code block to execute if condition is False
```
## Technologies Used
- Pycharm
- Python 3
## How to run
1. Clone the repository.
2. Open the project in pycharm.
3. Run the 'main.py' file.
4. View the result.
## Sample Output
```text
Enter Traffic light color: red

Stop
```
